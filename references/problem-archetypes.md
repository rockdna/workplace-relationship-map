# 六大经典职场关系困局

用户选场景时的参考库。每个场景包含：核心矛盾、常见变体、典型信号、易犯错误。

---

## 1. 🎯 向上管理

**核心矛盾**：你以为"做好事就行"，但上级要的可能是"让他安心"。

### 常见变体

| 变体 | 表面问题 | 真正的问题 |
|------|---------|-----------|
| 老板不表态 | 他不回消息、不说行不行 | 他不想担责，或者在等更好的方案出现 |
| 朝令夕改 | 昨天说A今天说B | 他自己也没想清楚，或者上级给他施压了 |
| 抢功 | 你的成果他汇报成了他的 | 他需要政绩，而你恰好是工具 |
| 不支持 | 资源不给、预算不批 | 他不信这件事能成，或者这件事对他的价值不清晰 |
| 微观管理 | 什么都管什么都改 | 他不信任你，或者他自己也焦虑 |

### 易犯错误
- 只说方案不说不做会怎样 → 老板没有"不做"的紧迫感
- 在公开场合反驳 → 老板在维护权威，不是在讨论对错
- 等 → "不表态"不是"不反对"，等下去他就找到不用你的理由了

---

## 2. 🤝 跨部门协作

**核心矛盾**：你以为"这是公司的事"，对方觉得"这不是我的事"。

### 常见变体

| 变体 | 表面问题 | 真正的问题 |
|------|---------|-----------|
| 推不动 | 对方不配合 | 你的优先级不是他的优先级 |
| 踢皮球 | "这个不归我们管" | 这个事做了对他没好处，不做也没坏处 |
| 对方优先级不同 | "我们这周忙" | 他不是没时间，是没动力给你时间 |
| 资源争夺 | 两边都要同一批人/预算 | 零和博弈，你多他少 |
| 信息不共享 | 对方不告诉你进展 | 他在保护自己的信息优势 |

### 易犯错误
- 用"公司目标"压人 → 对方听了只觉得"跟我有什么关系"
- 找对方上级施压 → 赢了这一次，下次他一定不配合
- 只说"我需要"不说"你能得到什么" → 利益没有对齐

---

## 3. ⚔️ 同级竞争

**核心矛盾**：你以为"实力说话"，但游戏规则可能是"站队说话"。

### 常见变体

| 变体 | 表面问题 | 真正的问题 |
|------|---------|-----------|
| 赛马 | 两团队做同一件事 | 上级在试，谁赢资源给谁——但你不知道规则 |
| 被边缘化 | 重要会议没你、关键信息不给你 | 有人系统性地在排挤你 |
| 功劳被抢 | 你做了他汇报 | 你没有"让上级看到"的机制 |
| 暗中使绊 | 看似配合实则拖延 | 他在等你失败，这样他就能上位 |
| 争上级注意 | 两个人争着表现 | 上级在享受这种竞争——但你在消耗 |

### 易犯错误
- 直接对峙 → 你没有证据，反而显得你小气
- 忍 → 他不会因为你忍了就停下来
- 找上级告状 → 除非你有铁证，否则上级觉得"你们自己解决"

---

## 4. 👥 向下管理

**核心矛盾**：你以为"我是领导自然听我的"，但团队服从的是权力之外的东西。

### 常见变体

| 变体 | 表面问题 | 真正的问题 |
|------|---------|-----------|
| 下属不服 | 公开质疑、消极抵抗 | 你没有赢得"他为什么听你的"这个前提 |
| 团队分裂 | 两派各听各的 | 团队里有一个非正式领袖，不是你 |
| 老员工摆烂 | 只做最低限度、不主动 | 他觉得不被重视，或者他的经验被忽略了 |
| 新人指挥不动 | 说了不做、做了打折 | 你还没有建立信任，你的指令在他看来是"又一个新官上任" |
| 中层架空 | 你说了他转达成另一个意思 | 他在用自己的理解改你的意图 |

### 易犯错误
- 用权力压 → 短期有效，长期他只做你盯着的事
- 跳过中层直接管基层 → 中层被架空，更不配合
- 只批评不给路 → "你这样做不对"但没有"你该怎么做"

---

## 5. 🏛️ 政治站队

**核心矛盾**：你以为"保持中立就行"，但中立本身就是一种选择——而且往往是最差的选择。

### 常见变体

| 变体 | 表面问题 | 真正的问题 |
|------|---------|-----------|
| 新旧势力 | 新领导来了旧团队不配合 | 你是旧团队的人，新领导在观察你的态度 |
| 领导更替 | 老领导走了新领导还没来 | 过渡期最危险——没人护你 |
| 夹缝生存 | 两个领导都要你站队 | 你选谁都不安全，但不选更不安全 |
| 选边压力 | "你到底跟谁？" | 他们在测试你的忠诚度 |
| 信息被封锁 | 你不在核心圈了 | 你已经被划到"另一边"了 |

### 易犯错误
- 以为中立安全 → 中立=两边都不信你
- 选边太早 → 那边可能输
- 选边太晚 → 赢的那边觉得你不够忠诚
- 跟错人 → 不要看权力看趋势——谁的上升势头更猛

---

## 6. 🔍 其他

用户自己描述。核心原则：
- 听他说什么，但判断他没说什么
- 如果描述模糊，追问一个具体例子
- 如果用户情绪强烈（愤怒、委屈、焦虑），先在回应中承认情绪，再进入分析——不要假装没看到

---

## 场景之间的交叉

真实困局经常跨越多个场景：

| 交叉 | 典型表现 |
|------|---------|
| 向上管理 + 跨部门 | 老板不支持，你需要跨部门自己搞 |
| 跨部门 + 同级竞争 | 两个部门做同一件事，赛马格局 |
| 向上管理 + 政治站队 | 新领导来了，你不知道站不站 |
| 向下管理 + 政治站队 | 团队里有旧势力的人，指挥不动 |
| 同级竞争 + 向上管理 | 同事比你更会向上管理 |

交叉场景的标注方式：主场景 + 次要维度。比如"跨部门协作（含赛马竞争）"。

## 7. 🎭 空降兵生存（新加入一个组织）

**核心矛盾**：你以为"履历够硬就能服众"，但老人们看的是"你是不是来抢我饭碗的"。

### 常见变体

| 变体 | 表面问题 | 真正的问题 |
|------|---------|-----------|
| 老人不服 | 公开质疑、消极抵抗 | 他们怕被边缘化，需要你给他们"安全感" |
| 老板不挺 | 出事没人撑腰 | 老板在观察你的"自救能力"——你越能自己搞定，他越愿意投资你 |
| 抓不到实权 | 签字权在老人手里 | 老人在用流程消耗你 |
| 招不到自己人 | 没人可用 | 你还没建立"基本盘" |
| 信息断流 | 老人不告诉你真话 | 你的"局"还没建立 |

### 易犯错误
- **急着立威** → 100天还没坐稳就立威，等于自掘坟墓
- **假装亲切** → 跟老人称兄道弟，他们不吃这一套
- **带自己人** → 空降带一群自己的人，老人的反扑会来得更猛
- **绕过老人直接找老板** → 老人会觉得你是"告密者"
- **什么都不做等 100 天** → 你在等的时候，别人在给你"上规矩"

### 黄金 100 天

```
第 1-30 天：调研期
  - 不出方案，只听、只看、只问
  - 1对1跟每个关键人聊 30 分钟（用"请教"的名义）
  - 识别三类人：盟友（10%）、中立（70%）、观望（20%）

第 31-60 天：小胜期
  - 选一个"低风险高收益"的小事，做出第一个胜利
  - 这个胜利必须让盟友看到"跟对人"，让观望看到"有戏"
  - 不要碰核心业务，先动边缘业务

第 61-100 天：扎根期
  - 把胜利变成"机制"（让胜利可复用）
  - 招 1-2 个自己人（必须从外部招，不要挖老人的人）
  - 第一次跟老板"要资源"——要的时候要具体、要得急
```

### 关键判断题
- 老人是"不服"还是"测试"？——测试你给他出题就接，不服你给他安全感
- 老板是"观察"还是"放弃"？——观察你还有戏，放弃你还在的话赶紧撤

## 8. 🐂 老黄牛困境（干最多活、没功劳）

**核心矛盾**：你以为"干得多自然被看见"，但**干的活越具体，你的可替代性越强**。

### 常见变体

| 变体 | 表面问题 | 真正的问题 |
|------|---------|-----------|
| 干活最多 | 加班最多但晋升轮不到 | 你被定位为"执行者"，不是"思考者" |
| 没存在感 | 做了很多事但没人知道 | 你没有"让上级看到"的机制 |
| 抢功无门 | 你的成果被汇报成别人的 | 你的工作流程里没有"留痕"和"汇报设计" |
| 老好人陷阱 | 谁找你都帮、最后自己累死 | 你不会说"不"，所以"烂活"都到你头上 |
| 工具人 | 你是"那个最能干的人" | 你是"那个最便宜的人"——能干的人不一定贵 |

### 易犯错误
- 以为"再努力一点就会被看见" → 你被看见的不是努力，是位置
- 主动包揽所有事 → 你越能干，越没人帮你
- 不汇报 → 做了 100 分的事，老板只看到 60 分
- 汇报太频繁 → 让老板觉得"你很闲"
- 抱怨不公平 → 抱怨=低价值信号，老板更不给你机会

### 老黄牛的三大破局点

**破局点 1：从"做事"到"立名"**
- 做事是被动消耗，立名是主动投资
- 立名 = 你的工作被外界知道、引用、复用的过程
- 三个动作：写总结、对外分享、命名方法论

**破局点 2：从"具体"到"抽象"**
- 具体的工作=易被替代
- 抽象的能力=稀缺资源
- 把你的"做事流程"提炼成"做事框架"，这个框架才值钱

**破局点 3：从"老好人"到"有边界"**
- 老好人不是善良，是"没有策略的退让"
- 学会说"我可以帮你做 A，但 B 你要自己搞定"
- 学会说"这周我已经满了，下周可以"

### 话术模板

**对老板的"立名"话术**：
> "我最近在整理XX项目的方法论，准备在公司内部做一个分享。**您觉得在什么场合讲比较合适？**"

**对同事的"立边界"话术**：
> "你这个事我特别想帮，但我这周已经被XX卡满了。**这样吧，下周一我专门花半天帮你梳理，你看行吗？**"

### 一句话诊断
> 你的活是不是**领导在汇报时主动提到**？如果半年内他没有一次主动提——你已经被定位为"工具人"。

## 9. 🚪 边缘人突围（重要会议没你、信息断流）

**核心矛盾**：你以为"是金子总会发光"，但**不发光的时候，别人以为你是石头**。

### 常见变体

| 变体 | 表面问题 | 真正的问题 |
|------|---------|-----------|
| 会议没你 | 重要决策都是小圈子定的 | 你的价值被低估，或者有人刻意不带你 |
| 信息断流 | 关键消息你总最后知道 | 你不在"信息链路"上 |
| 边缘项目 | 给你打杂活、不给主线任务 | 有人在保护自己的"主战场" |
| 集体忽视 | 团建、出差、活动都没你 | 你已经"被离职"了——只是还没通知 |
| 求助无门 | 出事没人帮你 | 你的"基本盘"已经空了 |

### 易犯错误
- 装作不知道 → 你越装，别人越认定你"无关紧要"
- 找老板告状 → 老板会觉得"你自己不会争取？"
- 直接质问同事 → 撕破脸但没解决根源
- 等别人想起你 → **永远不会等到的**

### 突围四步

**第一步：诊断"为什么被边缘"**
- 是**能力不够**？（真心需要补能力）
- 是**政治原因**？（有人不喜欢你）
- 是**位置问题**？（你的位置本身就在边缘）
- 是**风格问题**？（你太低调或者太张扬）
- 不同原因，策略不同

**第二步：找"破冰点"**
- 边缘人最忌讳"硬挤"——你硬挤进去大家更烦你
- 找一个小事主动做，做出**小亮点**
- 这个亮点必须让"信息圈"里的人看到

**第三步：找"内部人"**
- 边缘人最难的是没人替你说好话
- 必须在组织内找至少一个"内部人"——他愿意在合适场合提到你
- 怎么找内部人？找"有资源但不爱抢功"的人

**第四步：主动"汇报设计"**
- 你不设计汇报，别人不会自动发现你
- 每周/每月主动给老板发一份"工作进展邮件"
- 不需要长，3-5 个 bullet points 即可
- 关键是**让别人看到你的"思考过程"**，不是只看到"结果"

### 终极一问
> "如果我现在辞职了，公司要花多长时间才能找到一个替代我的人？"
> - 如果答案是"立刻" → 你已经深度边缘化
> - 如果答案是"3-6 个月" → 你还在局里，需要突围
> - 如果答案是"找不到" → 你没被边缘，是你自己感觉错了

## 10. 🌀 转型迷茫（天命已变，但你还在旧地图里）

**核心矛盾**：你以为"再撑一下就好了"，但**时代在抛弃你的时候，不会跟你打招呼**。

### 常见变体

| 变体 | 表面问题 | 真正的问题 |
|------|---------|-----------|
| 35 岁危机 | 年纪大了拼不动了 | 你的能力结构停在 25 岁 |
| 行业下行 | 越来越没前途 | 整个赛道在消失 |
| 技能过时 | 学不动了 | 你的"金箍棒"不再是金箍棒 |
| 温水煮青蛙 | 待遇还行但没意义 | 你的内驱力已经死了 |
| 错位感 | 我是不是做错行了 | 你的天赋/热情跟工作不匹配 |

### 易犯错误
- **硬撑** → 等着公司裁你
- **裸辞** → 没有准备就跳，往往跳得更差
- **假装热爱** → 假装三年后，你真的就"只能假装"了
- **换行业不换能力** → 换行业但不升级能力，只是换一种"困"的方式
- **完全听别人的** → 转型是自己的事，听多了反而乱

### 转型五问（自救指南）

**问 1：我是真"不喜欢"，还是"暂时不顺"？**
- 不喜欢 = 长期、深入、无法改变
- 不顺 = 短期、可能有解
- 区分不出来就先**做一件不花钱的"试"**——比如周末去做相关的事，看自己的真实反应

**问 2：我的"可迁移能力"是什么？**
- 不是"我能做什么工作"，是"我能做什么**类型**的事"
- 比如"我能从 0 到 1 创造新东西"——这个能力可迁移到创业、新业务、咨询
- 把"可迁移能力"列出来，再去找能用上的岗位

**问 3：我的"经济缓冲"够我撑多久？**
- 转型需要时间和钱
- 算清楚：你的存款够你**不工作**活多久？
- 如果不到 6 个月 → 边干边转，不要裸辞
- 如果超过 2 年 → 你可以更激进

**问 4：我能不能"渐进式转型"？**
- 最好的是**在公司内部转**：先在边缘做新事，等做起来了再正式转
- 第二好的是**边做边学**：保留当前工作，用业余时间试新方向
- 最差的是**一刀切**：裸辞 + 全新开始（高风险）

**问 5：我的"天命"到底是什么？**
- 天命不是"我想做什么"
- 天命是"即使没人给我钱，我也会做"的事
- 找不到天命没关系——**先做"小天命"**——一件你做的时候有感觉的小事
- 从小天命出发，慢慢看清大天命

### 转型期的心态管理
- **接受"过渡期"的存在** — 转型不是突变，是 2-3 年的过程
- **不要跟"还在 25 岁"的人比** — 你有你的优势：经验、判断、人脉
- **保持学习** — 转型期的"学习曲线"决定你能不能爬出来
- **保留旧世界的连接** — 转型时最蠢的是"把过去全部切断"
- **留出"撤退路线"** — 给自己的转型留 2-3 次失败的机会

### 一句话诊断
> "如果现在我中了 1000 万，我还会继续做现在的工作吗？"
> - 答案"会" → 不是转型问题，是心态问题
> - 答案"不会" → 是真转型问题，开始规划
> - 答案"不知道" → 你需要先想清楚"会做什么"再想转型
