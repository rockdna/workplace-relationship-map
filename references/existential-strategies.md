# 存在主义策略库 v1.1

> "人首先存在，然后定义自己。" — 萨特
>
> 职场上有些困境，不是技能问题、不是人脉问题、不是策略问题。是存在性问题——
> "我做这些是为了什么？""我是谁？""这是我的选择还是别人的期待？"
>
> 本策略库不是哲学课。是五个职场存在性困境的行动指南。

## 策略一：荒谬英雄

> **来源：加缪**《西西弗神话》《鼠疫》

### 第一层｜哲学叩问

加缪讲了一个故事：诸神惩罚西西弗，让他每天把巨石推到山顶，石头滚下来，再推上去，周而复始。

然后加缪说了一句让所有人愣住的话：**西西弗是幸福的。**

不是因为推石头有意义——推石头本身毫无意义。而是当西西弗意识到"这是我的命运，我选择继续推"时，他超越了诸神的惩罚。巨石是他的事了，跟诸神无关。

职场上：你的日报、周报、月度OKR、季度Review——年复一年。有时候你觉得这些石头推上去就是会滚下来的，什么都没变。**加缪不会骗你说"坚持就有意义"。他会问你：如果你知道这件事客观上没意义，你还做不做？如果做——你赋予它的意义是什么？**

### 第二层｜策略逻辑

意义不是你"找到"的——是你"赋予"的。

同样一份工作，有人觉得是"为了养家糊口"，有人觉得是"在学习行业规则"，有人觉得是"在积累客户关系"。**事情完全一样，意义完全不同。** 你选哪一个意义，决定了你的倦怠程度。

荒谬英雄的策略：不再问"这件事有没有意义"——改为问"我想从这件事里拿走什么"。

### 第三层｜具体招式

1. **重新定义你正在做的事**：写下你当前工作的三种不同定义——
   - 最功利的定义（"我就是为了攒钱"）
   - 最有成长的定义（"我在积累XX能力"）
   - 最个人的定义（"我在这里学会了怎么跟XX类型的人打交道"）
2. **选一个让你最不沮丧的定义**，用这个定义过接下来一个月。
3. **如果三个定义都让你沮丧**：不是工作的问题——你该换石头了。

**话术（给自己）**：
> "推石头本身没意义。但我选推这块石头，是为了XX。我是自由的。"

**风险**：不要把"赋予意义"理解成"自欺欺人"。你赋予的意义必须是真实的——不是骗自己"我很爱这份工作"，而是"这份工作确实让我拿到了XX"。

**🧭 追一问（给用户 2-3 个方向选，至少追两轮）**：

第一轮：
- A. "你说的'无聊'具体是哪一部分？是重复的内容，还是看不到成长，还是没人认可？"
- B. "如果这份工作明天消失，你最舍不得的会是什么？"

第二轮（根据用户第一轮回答追问）：
- 如果选 A 且说"重复" → "如果每周给你半天做完全不同的项目，你会选什么？"
- 如果选 A 且说"没成长" → "你最近一次觉得'学到了东西'是哪件事？"

## 策略二：自由之重

> **来源：萨特**《存在主义是一种人道主义》

### 第一层｜哲学叩问

萨特有一句让人很不舒服的话：**人被判定为自由。** 这不是祝福，是负担。

他的逻辑很残酷——你以为你没有选择："我只能待在这家公司""我只能接受这个安排""我只能忍到年底"。但萨特会说：不，每一次"不选择"，本身就是一次选择。你选择不跳槽、你选择不反抗、你选择等待——这些都是你选的。

更残酷的是：你不能把责任推给任何人。你的老板没有困住你——是你选择了留在这个老板手下。你的行业没有困住你——是你选择了不转行。

**萨特不是在指责你。他是在告诉你：如果你承认这些都是你的选择，那你也同时承认了——你可以做不同的选择。**

### 第二层｜策略逻辑

大多数人的职场困境不是"没有出路"，是"不敢承认出路的存在"。

- "我没法辞职，我有房贷" → 其实是"我选择用稳定的收入维持现在的房子，而不是换一个小房子换取自由"
- "我没法反抗老板" → 其实是"我选择忍受这个老板，因为辞职带来的不确定性比老板的压迫更让我恐惧"

**策略不是让你立刻辞职。是让你诚实地看清：你现在的处境，是你选出来的。** 你选它，一定有好理由。承认"这是我的选择"，比"我没得选"更有力量。

### 第三层｜具体招式

1. **把"我不得不"改成"我选择"**：写下三件你最近抱怨的事，把每句的"不得不"改成"我选择"。
   - "我不得不加班" → "我选择加班，因为___"
2. **写完那个"因为"**。你会看到你真正在权衡的是什么。
3. **如果那个"因为"依然成立**：你做了一个理性的选择。不再抱怨——因为这是你的选择。
4. **如果那个"因为"不成立了**：你该做不同的选择了。

**话术（给自己）**：
> "没有人把我关在这里。是我选择了留下，因为___。这个理由还成立吗？"

**风险**：自由之重可能导致"全怪自己"——不是所有困境都是你的选择。结构性不公、市场环境、歧视——这些不是你选的。自由之重适用于你真正有选择权的事。

**🧭 追一问（给用户 2-3 个方向选，至少追两轮）**：

第一轮：
- A. "那句'我不得不加班'，改写成'我选择加班，因为___'之后，那个'因为'是什么？"
- B. "如果你完全不管别人怎么想，你最想做的那个选择是什么？先别说出来，心里想就行。"

第二轮（根据用户第一轮回答追问）：
- 如果 A 的"因为"是怕 → "这个'怕'，有证据吗？还是你自己想象的？"
- 如果 B 用户迟疑 → "我没问你能不能做。我只问你最想做哪个。这两个问题是分开的。"

## 策略三：向死而生

> **来源：海德格尔**《存在与时间》

### 第一层｜哲学叩问

海德格尔有一个让哲学界震动的命题：人只有面对死亡时，才能真正地活着。

这个"死"在职场上不一定是真的死——是你的职业生涯的有限性。海德格尔会说：你知道你的岗位不可能永远在、你的行业不可能永远热、你的体力不可能永远撑得住。大多数职场焦虑的根源不是"我不够好"，是**你以为还有无限时间**——所以把真正重要的事一直往后推。

### 第二层｜策略逻辑

35岁危机、中年转型焦虑、职业下半场的迷茫——这些焦虑有一个共同结构：你终于意识到时间不是无限的。

"向死而生"的策略不是让你悲观。是让你用"有限性"来做决策：**如果你知道时间有限，那些"以后再说"的事就变成了"现在就得做"的事。**

这不是恐惧。是清醒。

### 第三层｜具体招式

1. **做一个"一年假设"练习**：假设你现在的岗位只能再做一年，这一年你要——
   - 做成哪一件事？（只一件）
   - 学会哪个能力？
   - 建立哪段关系？
2. **从下周一就开始做那件事**。不是在一年之后，是从下周一开始。
3. **每季度重复一次这个练习**。你会发现自己越来越专注在真正重要的事上。

**话术（给自己）**：
> "如果这岗位只剩一年，我不会把时间花在___上。所以，从下周开始，不做那个了。"

**风险**：不要变成"末日心态"——不是让你觉得反正没时间了所以焦虑。是让你用有限性来聚焦，而不是用来恐慌。

**🧭 追一问（给用户 2-3 个方向选，至少追两轮）**：

第一轮：
- A. "如果按'一年假设'，你选的那一件事是什么？"
- B. "你过去三个月花时间最多的事，跟你刚刚说的那件事——是一致的还是完全相反的？"

第二轮（根据用户第一轮回答追问）：
- 如果一致 → "那卡在哪？资源？授权？还是自己的犹豫？"
- 如果相反 → "那你每天花最多时间的那个事，是在给谁铺路？"

## 策略四：他者即地狱

> **来源：萨特**《禁闭》

### 第一层｜哲学叩问

萨特的名言"他人即地狱"可能是被误解最多的一句哲学金句。原意不是"别人都是坏人"——是：**当你通过别人的眼睛来看自己时，你就生活在地狱里。**

《禁闭》里，三个死人被关在一间房里，永远不能离开。他们互相观看、互相评判、互相定义。每个人都想通过另外两个人的眼睛确认"我是谁"，但另外两个人给的定义永远跟自己的感觉不一致。这就是地狱——不是火烤，是永远活在别人的目光里。

职场上：你以为你怕的是同事的议论、老板的评价、朋友圈的比较。但萨特会说：你真正怕的，是透过他们的眼睛看到的你自己——那个"不够好""不够快""不够强"的你。

### 第二层｜策略逻辑

只要你在通过别人的眼睛定义自己，你就永远在他人的地狱里——因为别人对你的看法永远跟你内心对自己的感觉不一致。

解决办法不是"不在乎别人怎么看"——那是反人性的。解决办法是：**意识到别人怎么看你是他的自由，你怎么看自己是你的自由。两个自由可以并存。**

### 第三层｜具体招式

1. **列出三个你最近最在意的"别人的评价"**。比如：
   - "同事可能觉得我能力不够"
   - "老板可能觉得我不够积极"
   - "同学觉得我混得没他好"
2. **对每条问**：这是他们说的，还是我猜的？如果是猜的，停——你在地狱活人的脑子里。
3. **如果是真有人说过**：那是他的看法。你的看法是什么？写下来。**两个看法可以不同，不同不意味着你是错的。**

**话术（给自己）**：
> "他怎么看我是他的自由。我怎么看我是我的自由。我不需要通过他的眼睛来确认我是谁。"

**风险**：不要走向"完全不在乎"——健康的做法是在乎但不被定义。你在乎同事的感受，不等于你要活成他们期待的样子。

**🧭 追一问（给用户 2-3 个方向选，至少追两轮）**：

第一轮：
- A. "你最在意的那个'别人怎么看我'——是你真听到的，还是你猜的？"
- B. "如果那个人明天离职，你们从此再无交集——你还这么在意他的看法吗？"

第二轮（根据用户第一轮回答追问）：
- 如果是猜的 → "那你等于在地狱一个活人的脑子里。你觉得你猜对了几成？"
- 如果是真听到的 → "那个人自己过得怎么样？他的人生是你想要的那种吗？"

## 策略五：本真之选

> **来源：克尔凯郭尔 / 海德格尔**

### 第一层｜哲学叩问

海德格尔说，大多数人活在"沉沦"状态——他们做的事、追求的东西、焦虑的方向，不是因为自己真正想要，是因为"别人都这样"。

克尔凯郭尔把这个问题推到更极致：当你面对一个真正属于自己的选择时——没有参考答案、没有人能帮你扛——你会焦虑到发抖。他说这叫"恐惧与战栗"。大多数人退回沉沦，不是因为懒，是因为那个焦虑太可怕了。

职场上：你追求的那个目标——晋升、高薪、大厂——是你自己选的，还是这个世界告诉你"成功应该长这样"？**克尔凯郭尔不会替你回答。他只会问：如果你不需要向任何人解释你的选择——你会选什么？**

### 第二层｜策略逻辑

"本真之选"不是你一觉醒来决定当艺术家然后辞职。它是一个持续的、日常的练习——每一次你问自己"这是我想要的还是别人说应该的"，你就在往前走一步。

不需要立刻推翻你全部的生活。从一件事开始：这件事，是你自己选的，跟别人无关。

### 第三层｜具体招式

1. **下周做一个"本真决策"**：
   - 选一件事——可以很小（中午吃什么是自己选的还是同事去哪我去哪）
   - 在决定之前问自己："如果世界上只有我一个人知道这件事的结果，没有人会评价我，我会怎么选？"
   - 按那个答案做一次。
2. **记录你的焦虑**：做完之后写下你的感受。大概率你会焦虑——那个焦虑就是克尔凯郭尔说的"发抖"。**感受它，但不要让感受替你决策。**
3. **积累"本真决策"的肌肉**：从小事开始，逐渐到大事。每一次都让你更清楚"我是谁"。

**话术（给自己）**：
> "这个选择——如果是世界上最后一个人类做的选择，没有人会评价——我会怎么选？"

**风险**：不要把"本真"当成"自私"——本真选择不是不顾他人，是忠于自己同时尊重他人。也不要把"本真"当成逃避责任的借口。

**🧭 追一问（给用户 2-3 个方向选，至少追两轮）**：

第一轮：
- A. "你最近做的哪个决定是'因为别人都这样所以我也这样'？能不能说出一个具体的？"
- B. "如果选大厂——3年后你在做什么？如果选创业——3年后你在哪里？哪个人生你更不想错过？"

第二轮（根据用户第一轮回答追问）：
- 如果 A 说不出 → "那我换个问法：你上一次'不听任何人的建议自己拍板'是什么时候？"
- 如果 B 选了但犹豫 → "你犹豫的那个理由，是你自己的还是别人的？"

## 何时用哪一策

```
你的困局是什么？
│
├─ 工作重复、麻木、觉得没意义 → 荒谬英雄
├─ 纠结要不要跳槽/转行/反抗但不敢动 → 自由之重
├─ 35岁危机、觉得时间不够了 → 向死而生
├─ 被同事评价/朋友圈比较压得喘不过气 → 他者即地狱
├─ 不知道自己真正想要什么、什么都是别人说"应该"的 → 本真之选
```
