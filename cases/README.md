# 案例库说明

> "修行不在别处，就在每一次困局里。" —— 悟空笔记

## 目录用途

存放**匿名化**的真实案例，用于：
1. 沉淀经验，避免重复踩坑
2. 反哺信号库（signal-patterns.md）和策略库（wukong-strategies.md）
3. 跨案例发现**新的模式**

## 命名规则

```
YYYY-MM-DD-场景名.md
```

示例：
- `2026-06-05-老黄牛困境.md`
- `2026-06-12-空降兵生存.md`
- `2026-06-20-跨部门协作.md`

## 匿名化规则

| 真实 | 替换为 |
|------|--------|
| 用户真实姓名 | 求问者 / A |
| 老板真实姓名 | 上级 / B |
| 同事真实姓名 | 甲 / 乙 / 丙 |
| 公司真实名称 | 某互联网公司 / 某科技公司 |
| 行业 | 科技 / 金融 / 制造 |
| 真实项目名 | 项目A / 项目B |

## 反哺机制

每次新增/更新案例后，检查：
- [ ] 是否出现**新信号**？→ 更新 `signal-patterns.md`
- [ ] 是否出现**新策略**或**现有策略需要调整**？→ 更新 `strategy-playbook.md` 或 `wukong-strategies.md`
- [ ] 是否有**跨场景通用模式**？→ 在 `SKILL.md` 中提炼

## 季度回顾

每季度（3个月/6个月/9个月/12月）做一次回顾：
- 统计**最高频场景**
- 统计**最有效策略**
- 整理**新发现信号**
- 整理**易犯错误清单**
